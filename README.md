## Development team

#### Nikita - Frontend developer

## Frontend

- Vue3 + VueRouter + Vuelidate
- TypeScript
- Pinia
- Axios
- SCSS

## Backend

- Django rest framework
- Celery
- Mailhog
- Redis
- Oauth2

## Запуск backend

1. cd backend/friendship-store
2. docker-compose build
3. docker-compose up

## Запуск frontend

1. cd frontend/
2. docker-compose build
3. docker-compose up
